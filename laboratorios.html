Laboratorios

Conoce nuestros modernos laboratorios equipados para la carrera de Diseño y Programación Web en el IEST Público "HUANTA"

🎓

Aula Pedagógica

Espacio diseñado para el aprendizaje interactivo y colaborativo, equipado con tecnología moderna para una experiencia educativa completa.
  • Mobiliario ergonómico y funcional
  • Pantallas interactivas de última generación
  • Proyectores multimedia de alta resolución
  • Routers MikroTik para conectividad
  • Sistema de sonido integrado
  • Iluminación LED ajustable
💻

Laboratorio de Desarrollo Web

Laboratorio especializado en desarrollo web con equipos de alto rendimiento y software profesional para formar desarrolladores competentes.
  • 40 PCs Lenovo i7, 16GB RAM, 1TB HDD
  • Software: Office 2019, VMware, VirtualBox
  • Sistemas: Windows/Ubuntu Server, Kaspersky
  • Hardware: SMART Board, proyector Epson
  • Impresora multifuncional
  • Router MikroTik y switches Kipands
🎨

Laboratorio de Diseño

Espacio creativo equipado con tecnología Apple y software profesional para diseño gráfico, desarrollo de aplicaciones y contenido multimedia.
  • 30 iMacs (Core i7), MacBook Pro
  • Tabletas Wacom, cámara Canon
  • Impresora 3D profesional
  • Software: Adobe Creative Suite, CorelDraw
  • Blender, plataformas e-commerce
  • Estación de fotografía profesional

La carrera de Diseño y Programación Web forma profesionales capaces de desarrollar soluciones tecnológicas con creatividad y eficiencia.
